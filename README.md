## LINER OSINT TOOL WEB APP 🌐⚡

**LINER OSINT TOOL APP** adalah sebuah dashboard web yang dirancang untuk mempercepat proses pengumpulan informasi (*information gathering*). Alat ini merangkum puluhan perintah *one-liner* tingkat lanjut, Google Dorks, dan skrip otomasi konsol ke dalam satu sistem manajemen yang bersih dan efisien.

Dikembangkan dan dikelola secara mandiri di bawah ekosistem **SPY-E & 123Tool**.

---

## 🛠️ Fitur Utama

*   **Interactive Terminal UI:** Tampilan bergaya konsol matriks dengan efek pendar (*glow*) dan *scanline* untuk kenyamanan investigasi jangka panjang.
*   **Instant Live Search:** Fitur pencarian dinamis yang menyaring perintah dork atau *one-liner* secara instan saat Anda mengetik.
*   **Multi-Category Navigation:** Pengelompokkan modul yang terstruktur:
    *   `PERSONAL_INFO` (Pelacakan data individu)
    *   `INFRASTRUCTURE_&_ORG` (Pemetaan jaringan & sub-domain korporat)
    *   `SOCMINT_&_NETWORKS` (Intelijen media sosial: X, Instagram, YT, dll.)
    *   `DARKWEB_&_LEAKS` (Pencarian sisa kebocoran data pada jaringan Tor)
    *   `GEOLOCATION_INTEL` (Ekstraksi metadata koordinat & pemetaan IP)
    *   `AUTOMATION_COMMANDS` (Skrip siap pakai untuk Sherlock, Amass, theHarvester, dll.)
    *   `PUBLIC_RECORDS_&_GOV` (Akses pangkalan data publik resmi)
*   **One-Click Copy:** Salin perintah rumit ke clipboard Anda secara instan tanpa risiko salah blokir teks.
